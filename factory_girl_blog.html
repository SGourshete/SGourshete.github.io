Factory Girl

Factories

1. Installing gem factory_girl_rails

$ gem install factory_girl_rails


2. Writing Factories

Consider a User model with attributes first_name, last_name, age, city, email_id.

FactoryGirl.define do

factory :user do

first_name 'Swapnil'
last_name 'Gourshete'
age 22
city 'Pune'
email example@example.com

end
end

3. Writing Traits

Consider additional attribute gender, which is stored in database as 'M' for Male and 'F' for Female.

FactoryGirl.define do
factory :user do
first_name 'Swapnil'
last_name 'Gourshete'

trait :male
gender 'M'
end

trait :female
gender 'F'
end

end
end

4. Writing with associations

Consider table locations with attributes user_id referencing to users table or User model, street, city, country.

FactoryGirl.define do

factory :location do

association(:user)
street '10 Downstreet'
city 'Manhatton'
country'United States'

end
end
